## Tamil-Kannada Script Comparison Notes (Feb 7, 2024)

```
அஆஇஈஉஊஎஏஐஒஓ
ಅಆಇಈಉಊಎಏಐಒಓ

ಕ க ಙ ங
ಚ ச ಞ ஞ
ಟ ட ಣ ண
ತ த ನ ந (voiced na)
ಪ ப ಮ ம
ಯ ய ರ ர ಲ ல ವ வ
ೞ ழ (like but not quite ಳ್ರ)
ಳ ள
ಱ ற (like ರ but tongue flaps multiple times like herding goats)
ನ್ ன (unvoiced na)
```

Notes: Tamil has ஐ and ஔ (= ಐ and ಔ). Tamil consonant ற (ಱ) sounds like ರ but tongue flaps multiple times.

## Etymology & Proto-Dravidian

Proto-Dravidian number reconstruction (personal notes, Jan 5, 2024):
```
ಒಣ್ರ್ -> ಒಣ್ಣ್/ಒಂದು (ಒಣ್/ಒರ್)
ಇರೆಂಡ್ರ್ -> ರೆಂಡ್/ಎರಡು (ಇರ್)
ಮೂಣ್ರ್ -> ಮೂಣ್/ಮೂರು (ಮೂರ್)
ನಾಳ್ರ್ -> ನಾಲ್/ನಾಲಕ್ಕು (ನಾಲ್)
ಚಯ್ಮ್ -> ಅಂಜ್/ಐದು (ಐ)
ಪತ್ತ್ -> ಪತ್ತ್/ಹತ್ತು (ಪತ್/ಹತ್)
ಒಣ್ + ಪತ್ತ್ -> ಒಂಬತ್ತು
ಇರ್ + ಪತ್ತ್ -> ಇರ್ಪತ್ತ್ -> ಇಪ್ಪತ್ತು
```
