# ಕನ್ನಡ Verb Paradigms

Verb paradigm tables for three representative Kannada verbs, following
D. N. Shankara Bhat — *ಕನ್ನಡ ಬರಹದ ಸೊಲ್ಲರಿಮೆ* Vol 2, §5.1.1.

Each entry has a plain text version (this site) and a **styled illustrated version**
with Noto Kannada fonts and colour-coded tense rows.

| Verb | Forms | Notes |
|------|-------|-------|
| [ಮಾಡು (mADu)](./mADu-verb-paradigm) | 36 | Regular ತಿರುಳೆಸಕಪದ — the reference paradigm |
| [ಇರು (iru)](./iru-verb-paradigm) | 42 | Suppletive stems; unique present tense |
| [ಇಲ್ಲ (illa)](./illa-verb-paradigm) | 5 | Invariant negator — minimal paradigm in Kannada |

**Reference:** D. N. Shankara Bhat, *ಕನ್ನಡ ಬರಹದ ಸೊಲ್ಲರಿಮೆ* Vol 2 ([Book 07](../dnsbhat/07-kannaDa-barahada-sollarime/))
