# Book 15 — Inglish-Kannada Padanerake: AI Context Primer

**For use by:** `ellara-kannada-word-coiner` skill, `kannada-morphology` skill, and any AI context that needs to coin or evaluate native Dravidian Kannada words.

**Book:** *Inglish-Kannada Padanerake* (English-Kannada Word Correspondence Dictionary)
**Authors:** D.N. Shankara Bhat, Y. Bharath Kumar, Vivek Shankar
**Published:** 2015, Navakarnataka Prakashan
**Pages:** x + 730 (sample: 53 pages, letter A only)

---

## What This Book Does

This dictionary maps ~5,000+ English words to **native Dravidian Kannada equivalents**, explicitly avoiding Sanskrit-derived substitutes. It is the most comprehensive single reference for native Kannada neologisms, and the primary practical companion to DNS Bhat's theoretical word-formation manual (Book 03, *Kannada Padagala Olaracane*).

**The dictionary's central claim:** For every English word, a native Kannada equivalent can be coined using existing Dravidian roots, Kannada compounding mechanisms, and Kannada derivational suffixes. No Sanskrit is required.

## Core Word-Formation Principles (for AI coin generation)

### Principle 1: Semantic compounding (N+N)

Join two native Kannada nouns: the **second noun is the head** (category), the **first noun is the modifier** (specifics).

```
ಗಾಳಿ (gALi, air) + ತೇರು (tEru, chariot) = ಗಾಳಿತೇರು (aircraft)
ಕೊಲ್ಲು (kollu, kill) + ಮನೆ (mane, house) = ಕೊಲ್ಲುಮನೆ (abattoir)
ಎಣ್ಣು (eNNu, count) + ಪಟ್ಟಿ (paTTi, board) = ಎಣ್ಣುಪಟ್ಟಿ (abacus)
ಎಚ್ಚರ (eccara, wakefulness) + ಗಂಟೆ (ganTe, bell) = ಎಚ್ಚರಗಂಟೆ (alarm clock)
```

### Principle 2: Verb-root + derivational suffix = noun

| Suffix | Function | Example |
|--------|----------|---------|
| **-ತ (-ta)** | action noun | ತೊರೆ (tore, forsake) → ತೊರೆತ (toreta, abandonment) |
| **-ಇಕೆ (-ike)** | process noun | ಹಂಚು (hancu, distribute) → ಹಂಚಿಕೆ (hancike, distribution) |
| **-ಉಗೆ (-uge)** | action result | ಕಿಚ್ಚಿಡು (kiccIDu, set fire) → ಕಿಚ್ಚಿಡುಗೆ (kiccIDuge, arson) |
| **-ಮೆ (-me)** | state/quality | ಅಡಕ (aDaka, compressed) → ಅಡಕಮೆ (aDakame, abbreviation) |
| **-ವು (-vu)** | verbal noun | ಎಯ್ತರು (eytaru, arrive) → ಎಳ್ತರವು (eLtaravu, arrival) |
| **-ಆಡು (-ADu)** | repetitive noun | ಏರ್ಪಡು (Erpadu, arrange) → ಏರ್ಪಾಡು (ErpADu, arrangement) |
| **-ಆಣಿಕೆ (-ANike)** | systemic noun | ನೀಗು (nIgu, nullify) → ನೀಗಾಣಿಕೆ (nIgANike, abolition) |

### Principle 3: Agent noun suffixes

| Suffix | Used with | Example |
|--------|-----------|---------|
| **-ಗ (-ga)** | short verb/noun stem | ಪೂಣಿ (pUNi) → ಪೂಣಿಗ (pUNiga, archer) |
| **-ಉಗ (-uga)** | verb phrase | ಹಾರಿಸು (hArisu) → ಹಾರಿಸುಗ (hArisuga, abductor) |
| **-ಗಾರ (-gAra)** | abstract noun base | ತೀರ್ಪು (tIrpu) → ತೀರ್ಪುಗಾರ (tIrpugAra, arbitrator) |

### Principle 4: ಅರಿಮೆ (arime) compounds for academic disciplines

*arime* (from root *ari* = to know) = any systematic field of knowledge.

**Formula:** [domain-descriptor] + ಅರಿಮೆ

```
ಇರ್ಪರಿಮೆ = matter-science = chemistry
ಗಾಳಿಯೊತ್ತರಿಮೆ = air-pressure-science = pneumatics
ಪಳಮೆಯರಿಮೆ = ancient-science = archaeology
ಲೆಕ್ಕದರಿಮೆ = calculation-science = arithmetic
ಬಿತ್ತೆಣಿಕೆಯರಿಮೆ = variable-arithmetic-science = algebra
```

Use this to coin names for any academic discipline or professional field.

## Part-of-Speech Framework

The dictionary uses Bhat's own Kannada grammatical terms:

| Abbr. | Kannada | Eke | English |
|-------|---------|-----|---------|
| ಹೆ | ಹೆಸರುಪದ | hesarupada | noun |
| ಎ | ಎಸಕಪದ | esakapada | verb |
| ಪ | ಪರಿಚೆಪದ | paricepada | adjective/qualifier |

**Key rule:** The part-of-speech label applies to the **Kannada word**, not the English headword. An English noun can be given a Kannada verb equivalent and vice versa.

## Word-Generation Decision Tree for the Coiner

When asked to coin a Kannada native word for an English concept, use this order of preference:

1. **Check if a native Kannada word already exists** (Kittel's dictionary, Dravidian Etymological Dictionary). Use it directly with new meanings if needed.

2. **Identify the semantic domain** — does the concept involve:
   - Air/sky? → Use ಗಾಳಿ- prefix cluster
   - Knowledge/science? → Use -ಅರಿಮೆ suffix
   - House/institution? → Use -ಮನೆ head
   - Treatment/therapy? → Use -ಮಾಂಜುಗೆ head
   - Transport? → Use -ಸಾರಿಗೆ, -ತೇರು, -ರೇವು heads
   - Water? → Use ನೀರ- prefix cluster
   - Old/ancient? → ಪಳ-, ಹಳೆ- prefix

3. **If it's an action concept:** Find the native Kannada verb root for the action. Add -ತ, -ಇಕೆ, or -ಉಗೆ to nominalize.

4. **If it's a person concept (agent):** Use verb stem + -ಗ, -ಉಗ, or -ಗಾರ.

5. **If it's a discipline name:** [core concept in Kannada] + ಅರಿಮೆ.

6. **If it's a compound concept:** Identify the category (head) and the differentiator (modifier). [modifier] + [head].

7. **Verify Dravidian character:** The result should sound Kannada — it will have:
   - Words beginning with ಅ, ಎ, ಒ (not ಸ್ರ, ಪ್ರ, ಅನ್ as Sanskrit often does)
   - Native retroflex sounds (ಟ, ಡ, ಣ, ಳ) are OK
   - No aspirates (ಥ, ಭ, ಫ, ಧ etc.) — these are Sanskrit indicators
   - Compound structure that feels transparent and motivatable

## Common Mistakes to Avoid

| Mistake | Wrong | Right |
|---------|-------|-------|
| Using Sanskrit suffix -na/-ana | ಬಿಡುಗಾಶ್ರಮ (bIDugASrama) | ಬಿಡುಗಮನೆ (biDugamane) |
| Using ಶಾಸ್ತ್ರ (SAstra) for -ology | ಪುರಾತತ್ವಶಾಸ್ತ್ರ | ಪಳಮೆಯರಿಮೆ (paLameyarime) |
| Sanskrit prefix ಪ್ರ- | ಪ್ರಸರಣ (prasaraNa) | ಹರವು (haravu) / ಹರಡಿಕೆ |
| Sanskrit suffix -ಣ (-Na) | ಭೋಜನ (bOjana) | ಊಟ (UTa) / ತಿನಿಸು (tinisu) |
| Sanskrit compound ಮಹಾ- | ಮಹಾಪ್ರಾಣ | (full Kannada term for aspirate concept) |
| Sanskrit -ಕ (-ka) agent | ವಾಹಕ (vAhaka) | ಸಾಗಿಸುಗ (sAgisuga) |

## The Philosophical Frame (for context-setting)

This dictionary embodies three convictions that should guide any Kannada word-coiner:

1. **ಕನ್ನಡಕ್ಕೆ ಶಕ್ತಿಯಿದೆ** — Kannada has the power. It has roots for almost every concept; they just need to be extended into new domains.

2. **ಸಂಸ್ಕ್ರುತ ಅನಿವಾರ್ಯವಲ್ಲ** — Sanskrit is not necessary. Every time a Kannada writer uses Sanskrit for a technical concept, a native Kannada word loses a chance to grow.

3. **ಎಲ್ಲರ ಕನ್ನಡ** — Kannada for everyone. The best Kannada word is the one that a Kannada speaker can understand on hearing it, without specialised Sanskrit learning. Transparency and motivatedness are valued over prestige.
