# 29 — ಕನ್ನಡ ವ್ಯಾಕರಣ ಯಾಕೆ ಬೇಕು?
**Why Do We Need Kannada Grammar?**

> Makes the motivating case for studying Kannada grammar — who needs it, why, and how it must differ from Sanskrit grammar. Aimed at skeptical general readers. Written in *hosa baraha* (Bhat's simplified orthography without mahaprana letters). Third book in the popular advocacy trilogy; where Book 28 argues *what kind* of grammar Kannada needs, this book argues *why* grammar matters at all.
