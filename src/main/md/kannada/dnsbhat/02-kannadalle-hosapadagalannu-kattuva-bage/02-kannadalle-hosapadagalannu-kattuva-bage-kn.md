## ಭಾಗ ೧ — ಮುನ್ನುಡಿ — ಕನ್ನಡದ ಒಲವುಗಳು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

<a id="sec-1-1"></a>

### ೧.೧ ಬರಹ ಕನ್ನಡದಲ್ಲಿ ಸಂಸ್ಕೃತದ ಪ್ರಾಬಲ್ಯ

ಕನ್ನಡದ ಬರಹಗಾರರು ಮತ್ತು ಅರಿವಿಗರು ಹೊಸ ಪದಗಳನ್ನು ಕಟ್ಟುವಾಗಲೆಲ್ಲ ಸಂಸ್ಕೃತ ಪದ ಮತ್ತು ಒಟ್ಟುಗಳನ್ನು ಬಳಸುವುದೇ ಹೆಚ್ಚು; ಕನ್ನಡದ ತನ್ನದೇ ಆದ ಪದ ಮತ್ತು ಒಟ್ಟುಗಳನ್ನು ಬಳಸಿಯೂ ಹೊಸ ಪದಗಳನ್ನು ಕಟ್ಟಬಹುದು ಎಂಬ ಅರಿವು ಅವರಿಗಿರುವುದಿಲ್ಲ. ಇದರ ಪರಿಣಾಮವಾಗಿ ಕನ್ನಡದ ಮೂಲ ಪದಗಳು ಒಂದೊಂದಾಗಿ ಬರಹದ ಬಳಕೆಯಿಂದ ಹೊರಬೀಳುತ್ತಿದ್ದು, ಅವುಗಳ ಜಾಗವನ್ನು ಸಂಸ್ಕೃತ ಎರವಲುಗಳು ತುಂಬುತ್ತಿವೆ. ಬಳಕೆಯಲ್ಲಿರುವ ಕನ್ನಡ ಪದನೆರಕೆಗಳಲ್ಲಿ ನೂರಕ್ಕೆ ಐವತ್ತಕ್ಕಿಂತ ಹೆಚ್ಚು ಸಂಸ್ಕೃತ ಎರವಲುಗಳಿದ್ದು, ಅರಿಮೆ ವಿಷಯಗಳ ನೆರಕೆಗಳಲ್ಲಿ ಇದು ನೂರಕ್ಕೆ ಎಂಬತ್ತರಷ್ಟಕ್ಕೆ ಏರುತ್ತದೆ. ಸಂಸ್ಕೃತ ಎರವಲುಗಳ ಈ ಹೊರೆಯಿಂದಾಗಿ ಕನ್ನಡ ಬರಹ ಆಡುನುಡಿಯಿಂದ ದೂರ ಸರಿದು ತನ್ನತನವನ್ನು ಕಳೆದುಕೊಳ್ಳುತ್ತಿದೆ.

<a id="sec-1-2"></a>

### ೧.೨ ಪರಿಹಾರ — ಕನ್ನಡದ ನಿಜ ಪದ ಕಟ್ಟುವ ತಾಕತ್ತು

ಕನ್ನಡದ ತನ್ನದೇ ಆದ ಪದ ಮತ್ತು ಒಟ್ಟುಗಳನ್ನು ಬಳಸಿ ಹೊಸ ಪದಗಳನ್ನು ಕಟ್ಟಲು ಬರುತ್ತದೆ ಎಂಬ ಸತ್ಯ ಹೆಚ್ಚಿನ ಬರಹಗಾರರಿಗೆ ತಿಳಿದಿಲ್ಲ. ಹೆಚ್ಚು ಹೆಚ್ಚು ಕನ್ನಡ ಪದಗಳನ್ನು ಬರಹದಲ್ಲಿ ಬಳಸಬೇಕು, ಕನ್ನಡ ಬರಹ ಎಲ್ಲ ಬಗೆಯ ಕನ್ನಡಿಗರನ್ನು ತಲುಪಬೇಕು ಎಂಬ ತುಡಿತ ಇಂದು ಹಲವರಲ್ಲಿ ಮೊಳೆತಿದೆ. ಈ ಒಲವಿಗೆ ಮೂರ್ತ ರೂಪ ಕೊಡಬೇಕಾದರೆ ಕನ್ನಡ ಪದ ಕಟ್ಟುವ ಒಲವುಗಳನ್ನು ಅರಿತು ಹೊಸ ಪದಗಳನ್ನು ತಾವೇ ಕಟ್ಟುವ ಕೆಲಸಕ್ಕೆ ಇಳಿಯಬೇಕಾಗಿದೆ.

<a id="sec-1-3"></a>

### ೧.೩ ಪುಸ್ತಕದ ಉದ್ದೇಶ ಮತ್ತು ವ್ಯಾಪ್ತಿ

ಈ ಕಡತ ಕನ್ನಡದಲ್ಲೇ ಹೊಸ ಪದಗಳನ್ನು ಕಟ್ಟಬೇಕೆಂದಿರುವವರಿಗೆ ಬೇಕಾದ ತಿಳಿವನ್ನು ಒದಗಿಸಿಕೊಡುವ ಮುಖ್ಯ ಉದ್ದೇಶ ಹೊಂದಿದೆ. ಮೊದಲ ತುಂಡಿನಲ್ಲಿ ಹೊಸ ಪದಗಳನ್ನು ಕಟ್ಟುವಲ್ಲಿ ಕನ್ನಡದ ಒಲವುಗಳು ಎಂತಹವು ಎಂಬುದನ್ನು ವಿವರಿಸಲಾಗಿದ್ದು, ಎರಡನೇ ತುಂಡಿನಲ್ಲಿ ಇಂಗ್ಲಿಶ್ ಪದಗಳಿಗೆ ಸಾಟಿಯಾಗಿ ಕನ್ನಡ ಪದಗಳನ್ನು ಕಟ್ಟುವ ಬಗೆಯನ್ನು ತೋರಿಸಿಕೊಡಲಾಗಿದೆ. ಕನ್ನಡ ಬರಹ ಎಲ್ಲ ಕನ್ನಡಿಗರಿಗೂ ಹತ್ತಿರವಾಗುವಂತೆ ಮಾಡುವಲ್ಲಿ ಮತ್ತು ಕನ್ನಡ ಪದಗಳ ಅಳವನ್ನು ಹೆಚ್ಚಿಸುವಲ್ಲಿ ಈ ಕಡತ ನೆರವಾಗಲಿ ಎಂಬ ಹಾರೈಕೆ ಲೇಖಕರದ್ದು.

### ೨.೧ ಮೂರು ಪದ ವರ್ಗಗಳು

ಕನ್ನಡದಲ್ಲಿ ಮುಖ್ಯವಾಗಿ ಮೂರು ಬಗೆಯ ಪದಗಳು ಬಳಕೆಯಲ್ಲಿವೆ: ಹೆಸರು ಪದ (ನೌನ್), ಎಸಕ ಪದ (ವರ್ಬ್), ಮತ್ತು ಪರಿಚಯ ಪದ (ಅಡ್ಜೆಕ್ಟಿವ್/ಅಡ್ವರ್ಬ್). ಹೆಸರು ಪದಗಳು ಮಂದಿ, ಪುರುಳು, ಪರಿಚೆ, ಎಸಕ ಮೊದಲಾದ ಪಾಂಗುಗಳನ್ನು ಹೆಸರಿಸುತ್ತವೆ; ಎಸಕ ಪದಗಳು ಯಾವುದಾದರೊಂದು ಯಸಕ ನಡೆಯುತ್ತಿರುವುದನ್ನು ತಿಳಿಸುತ್ತವೆ; ಪರಿಚಯ ಪದಗಳು ಹೆಸರು ಪದ ಅಥವಾ ಎಸಕ ಪದದ ಪರಿಚಯ ನೀಡುತ್ತವೆ. ಈ ಮೂರೂ ಬಗೆಯ ಪದಗಳನ್ನು ಹೊಸದಾಗಿ ಕಟ್ಟಿ ಕನ್ನಡ ಪದ ಅಳವನ್ನು ಹಿಗ್ಗಿಸಬಹುದು.

<a id="sec-2-2"></a>

### ೨.೨ ಎರಡು ವಿಧಾನಗಳು — ಕಟ್ಟುಪದ ಮತ್ತು ಜೋಡುಪದ

ಕನ್ನಡದಲ್ಲಿ ಹೊಸ ಪದಗಳನ್ನು ಕಟ್ಟಲು ಮೂಲತಃ ಎರಡು ಹಮ್ಮುಗೆಗಳಿವೆ. ಮೊದಲ ಹಮ್ಮುಗೆ: ಒಂದು ಪದಕ್ಕೆ ಒಟ್ಟನ್ನು ಸೇರಿಸಿ ಕಟ್ಟು ಪದ ಮಾಡುವುದು — ಉದಾ: ಓದು + ಗ = ಓದುಗ. ಎರಡನೇ ಹಮ್ಮುಗೆ: ಎರಡು ಪದಗಳನ್ನು ಒಂದಕ್ಕೊಂದು ಸೇರಿಸಿ ಜೋಡು ಪದ ಮಾಡುವುದು — ಉದಾ: ಮನೆ + ಕೆಲಸ = ಮನೆಕೆಲಸ. ಎಸಕ ಪದಗಳಿಗೆ ಇನ್ನೊಂದು ಎಸಕ ಪದವನ್ನು ಜೋಡಿಸುವ ರೂಪದಲ್ಲಿ ಸೇರಿಸಿ ಕೂಡು ಪದ ಮಾಡುವ ಹಮ್ಮುಗೆಯೂ ಕನ್ನಡದಲ್ಲಿ ಹೇರಳವಾಗಿ ಬಳಕೆಯಲ್ಲಿದೆ.

<a id="sec-2-3"></a>

### ೨.೩ ನಾಲ್ಕು ಮುನ್ನೊಟ್ಟು ಗುಂಪುಗಳು

ಕನ್ನಡ ಪದ ಕಟ್ಟಿಕೆಯ ಚೌಕಟ್ಟಿನಲ್ಲಿ ಮುನ್ನೊಟ್ಟು ಬೇರುಗಳನ್ನು ಅವು ತಿಳಿಸುವ ಹುರುಳಿನ ಮೇಲೆ ನಾಲ್ಕು ಗುಂಪುಗಳಲ್ಲಿ ಅಣಿಗೊಳಿಸಬಹುದು. ಮೊದಲನೆಯದು ಅಳವಿಯ ಮುನ್ನೊಟ್ಟು — ಎಣಿಕೆ ಅಥವಾ ಹಂತ ತಿಳಿಸುವ ಬೇರುಗಳು (ಒರ್-, ಈರ್-, ಮಿಗಿಲ-); ಎರಡನೆಯದು ಇಂಬಿನ ಮುನ್ನೊಟ್ಟು — ಸ್ಥಳ ಮತ್ತು ದಿಕ್ಕು ತಿಳಿಸುವ ಬೇರುಗಳು (ಮುನ್-, ಹೊರ-, ಒಳ-); ಮೂರನೆಯದು ಹೊತ್ತಿನ ಮುನ್ನೊಟ್ಟು — ಕಾಲ ತಿಳಿಸುವ ಬೇರುಗಳು (ಹಿಂ-, ಮುನ್-); ನಾಲ್ಕನೆಯದು ಅಲ್ಲಗಳೆಯುವ ಮುನ್ನೊಟ್ಟು — ಇಲ್ಲ/ಅಲ್ಲ ಎಂಬ ಅರ್ಥ ಕೊಡುವ ಬೇರುಗಳು. ಈ ನಾಲ್ಕು ಗುಂಪಿನ ಬೇರುಗಳು ಇಂಗ್ಲಿಶ್ ಮುನ್ನೊಟ್ಟುಗಳ ಕನ್ನಡ ಸಾಟಿಗಳಾಗಿ ಬಳಕೆ ಆಗಬಲ್ಲವು.

---


<a id="ch3"></a>

## ಭಾಗ ೩ — ಪರಿಚೆಪದದಿಂದ ಹೆಸರುಪದ

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-3-1"></a>

### ೩.೧ ಗುಣ ಒಟ್ಟು -ತನ

ಪರಿಚಯ ಪದ (ಅಡ್ಜೆಕ್ಟಿವ್)ಗಳಿಂದ ಅಮೂರ್ತ ಗುಣ ತಿಳಿಸುವ ಹೆಸರು ಪದಗಳನ್ನು ಕಟ್ಟಲು ಕನ್ನಡದಲ್ಲಿ -ತನ ಎಂಬ ಒಟ್ಟು ಬಳಕೆಯಾಗುತ್ತದೆ; ಇದು ಇಂಗ್ಲಿಶ್ -ness/-ity ಒಟ್ಟುಗಳ ಕನ್ನಡ ಸಾಟಿ. ಉದಾ: ದಿಟ್ಟ → ದಿಟ್ಟತನ, ಕಹಿ → ಕಹಿತನ, ದೊಡ್ಡ → ದೊಡ್ಡತನ, ಗಟ್ಟಿ → ಗಟ್ಟಿತನ. ಕನ್ನಡದ ಯಾವ ಪರಿಚಯ ಪದಕ್ಕೂ -ತನ ಒಟ್ಟನ್ನು ಸೇರಿಸಿ ಅಮೂರ್ತ ಹೆಸರು ಪದ ಕಟ್ಟಬಹುದು; ಇದು ಕನ್ನಡದ ಒಂದು ತುಂಬ ಬಲವಾದ ಪದ ಕಟ್ಟಿಕೆ ಒಲವು.

<a id="sec-3-2"></a>

### ೩.೨ ತಿರುಗು ರೀತಿ — ಹೆಸರುಪದದಿಂದ ಪರಿಚೆಪದ

ತಿರುಗು ರೀತಿಯಲ್ಲಿ ಹೆಸರು ಪದದಿಂದ ಪರಿಚಯ ಪದ ಕಟ್ಟುವ ಅಗತ್ಯ ಬಂದಾಗ ಕನ್ನಡದಲ್ಲಿ ಹೆಸರು ಪದದ ಪತ್ತುಗೆ ರೂಪ (-ದ/-ಯ ಒಟ್ಟು) ಬಳಸಲಾಗುತ್ತದೆ. ಉದಾ: ಒಲುಮೆ → ಒಲುಮೆಯ (loving), ಕೆಚ್ಚು → ಕೆಚ್ಚಿನ, ಗೆಳೆತನ → ಗೆಳೆತನದ. ಇದಕ್ಕೆ ಹೆಚ್ಚಿನ ಹುರುಳನ್ನು ಸೇರಿಸಬೇಕಾದಾಗ ಉಳ್ಳ (full of), ಇಲ್ಲದ (without), ತನ (quality of) ಮೊದಲಾದ ಒಟ್ಟುಗಳನ್ನು ಬಳಸಲು ಬರುತ್ತದೆ.

---


<a id="ch4"></a>

## ಭಾಗ ೪–೫ — ಎಸಕಪದದಿಂದ ಹೆಸರುಪದ

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-4-1"></a>

### ೪.೧ ಎಸಗುಗ ಒಟ್ಟು -ಗ/-ಇಗ

ಮಂದಿಯನ್ನು ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ ಮುಖ್ಯವಾಗಿ -ಗ/-ಇಗ ಎಂಬ ಒಟ್ಟನ್ನು ಬಳಸಲಾಗುತ್ತದೆ. ಉಕಾರದಲ್ಲಿ ಕೊನೆಗೊಳ್ಳುವ ಎಸಕ ಪದಗಳ ಬಳಿಕ -ಗ/-ಇಗ ಸೇರಿ ಎಸಗುಗ ರೂಪ ಪಡೆಯುತ್ತದೆ — ಉದಾ: ಓದು → ಓದುಗ, ಬೇಡು → ಬೇಡುಗ. ಈ ಒಟ್ಟನ್ನು ಹೆಸರು ಪದಗಳಿಗೂ ಸೇರಿಸಬಹುದು — ಉದಾ: ಗಾಣ → ಗಾಣಿಗ, ನಾಡು → ನಾಡಿಗ, ಕೆಚ್ಚು → ಕೆಚ್ಚುಗ. ಹೀಗೆ ಕಟ್ಟಿದ ಪದಗಳಲ್ಲಿ ಯಾವ ಎಸಕ ನೇರ ಹೇಳಿಲ್ಲದಿದ್ದರೂ ಒಂದು ಯಸಕ ಅಡಗಿಕೊಂಡಿರುತ್ತದೆ.

<a id="sec-4-2"></a>

### ೪.೨ ವೃತ್ತಿ ಎಸಗುಗ ಒಟ್ಟು -ಗಾರ

ಮಂದಿಯನ್ನು ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ ಹೆಚ್ಚು ಬಳಕೆಯಲ್ಲಿರುವ ಇನ್ನೊಂದು ಒಟ್ಟು -ಗಾರ. ಇದನ್ನು ಹೆಸರು ಪದಗಳ ಬಳಿಕ ಮಾತ್ರ ಸೇರಿಸಲಾಗುತ್ತದೆ; -ಗ/-ಇಗ ಒಟ್ಟಿನ ಹಾಗೆ ಎಸಕ ಪದಗಳ ಬಳಿಕ ಬರುವುದಿಲ್ಲ. ಉದಾ: ದೂರು → ದೂರುಗಾರ, ಮೀನು → ಮೀನುಗಾರ, ಓಟ → ಓಟಗಾರ, ಸಾಲ → ಸಾಲಗಾರ, ಮೆಟ್ಟು → ಮೆಟ್ಟುಗಾರ. -ಗಾರ ಒಟ್ಟಿನ ಪದಗಳಲ್ಲಿ ಆ ಹೆಸರು ಪದ ಗುರುತಿಸುವ ಪಾಂಗನ್ನು ಮುಖ್ಯ ಪರಿಚಯ ಆಗಿ ಪಡೆದಿರುವ ಮಂದಿಯನ್ನು ಹೆಸರಿಸಲಾಗುತ್ತದೆ.

<a id="sec-4-3"></a>

### ೪.೩ ಒಳಗಾಗುಗ ಸಮಸ್ಯೆ (-ee ಅಂತರ)

ಇಂಗ್ಲಿಶ್ನಲ್ಲಿ -ee ಒಟ್ಟು ಒಂದು ಯಸಕಕ್ಕೆ ಒಳಗಾದ ಮಂದಿಯನ್ನು (ಪಡೆಗ) ಹೆಸರಿಸುತ್ತದೆ — ಉದಾ: trainee, employee, interviewee. ಕನ್ನಡದಲ್ಲಿ ಇಂತಹ ಒಂದು ನಿರ್ದಿಷ್ಟ ಒಟ್ಟಿಲ್ಲ; ಹಾಗಾಗಿ ಇಂಗ್ಲಿಶ್ -ee ಒಟ್ಟಿರುವ ಪದಗಳಿಗೆ ಸಾಟಿ ಕನ್ನಡ ಪದ ಕಟ್ಟಲು ಕಷ್ಟ. ಇಂತಹ ಕಡೆಗಳಲ್ಲಿ ಪಡೆಗ ಎಂಬ ಪದ ಅಥವಾ ಕೂಡು ಪದ ರೂಪಗಳನ್ನು ಬಳಸಬಹುದು — ಉದಾ: ಹಣ ಪಡೆಗ (payee), ಸಂದರ್ಶನ ಪಡೆಗ (interviewee).

<a id="sec-4-4"></a>

### ೪.೪ ಉಪಕರಣ ಒಟ್ಟು -ಕ

ಒಂದು ಯಸಕ ನಡೆಸಲು ಬಳಸಲಾಗುವ ಮುಟ್ಟು (ಉಪಕರಣ) ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ -ಕ ಎಂಬ ಒಟ್ಟನ್ನು ಸೇರಿಸಲಾಗುತ್ತದೆ. ಉದಾ: ಸುಲಿ → ಸುಲಿಕ (peeler), ತಿಕ್ಕು → ತಿಕ್ಕಕ (scrubber), ಅಚ್ಚು ಹಾಕು → ಅಚ್ಚುಗಾರ (printer). ಇಂಗ್ಲಿಶ್ -er/-or ಒಟ್ಟು ಮುಟ್ಟನ್ನು ಹೆಸರಿಸಲು ಬಳಕೆ ಆದಾಗ ಅದರ ಕನ್ನಡ ಸಾಟಿಯಾಗಿ -ಕ ಒಟ್ಟನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.

<a id="sec-4-5"></a>

### ೪.೫ ಅಮೂರ್ತ ಎಸಕ ಹೆಸರುಪದ ಒಟ್ಟು -ಇಕೆ/-ಕೆ ಮತ್ತು -ತ

ಎಸಕ ಪದ ತಿಳಿಸುವ ಯಸಕವನ್ನು ಹೆಸರಿಸಲು (ಅಮೂರ್ತ ಹೆಸರು ಪದ ಕಟ್ಟಲು) ಕನ್ನಡದಲ್ಲಿ ಮುಖ್ಯವಾಗಿ ಎರಡು ಒಟ್ಟುಗಳನ್ನು ಬಳಸಲಾಗುತ್ತದೆ. -ಇಕೆ/-ಕೆ ಒಟ್ಟು ಉಕಾರದಲ್ಲಿ ಕೊನೆಗೊಳ್ಳುವ ಎಸಕ ಪದಗಳ ಬಳಿಕ ಬರುತ್ತದೆ — ಉದಾ: ಅನಿಸು → ಅನಿಸಿಕೆ, ಬೆದರು → ಬೆದರಿಕೆ, ಹರಡು → ಹರಡಿಕೆ, ಅಗ್ಗಳಿಸು → ಅಗ್ಗಳಿಕೆ. -ತ ಒಟ್ಟು ಆಕಾರ ಅಥವಾ ಎಕಾರದಲ್ಲಿ ಕೊನೆಗೊಳ್ಳುವ ಎಸಕ ಪದಗಳ ಬಳಿಕ ಬರುತ್ತದೆ — ಉದಾ: ತಿನ್ನು → ತಿಂತ, ಓಡು → ಓಟ.

<a id="sec-4-6"></a>

### ೪.೬ ಮೂರ್ತ ಫಲ ಹೆಸರುಪದ ಒಟ್ಟು -ಗೆ/-ಇಗೆ ಮತ್ತು -ತೆ

ಎಸಕದ ಮೂರ್ತ ಫಲ ಅಥವಾ ದೊರೆತ ಪಾಂಗನ್ನು ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ -ಗೆ/-ಇಗೆ ಮತ್ತು -ತೆ ಎಂಬ ಒಟ್ಟುಗಳನ್ನು ಬಳಸಲಾಗುತ್ತದೆ. -ಗೆ/-ಇಗೆ ಬಳಕೆ: ಬರಿ → ಬರಿಗೆ (stroke/writing), ಹೊಲಿ → ಹೊಲಿಗೆ (stitching). -ತೆ ಬಳಕೆ: ಬಿಡು → ಬಿಡುಗಡೆ (release), ಕೊಡು → ಕೊಡುಗೆ (gift/result of giving). -ಇಕೆ/-ಕೆ, -ತ ಒಟ್ಟುಗಳು ಅಮೂರ್ತ ಯಸಕ ಹೆಸರಿಸುತ್ತವೆ; -ಗೆ/-ಇಗೆ, -ತೆ ಒಟ್ಟುಗಳು ಮೂರ್ತ ಫಲ ಅಥವಾ ದೊರೆತ ಪಾಂಗ ಹೆಸರಿಸುತ್ತವೆ.

## ಭಾಗ ೬ — ಶೂನ್ಯ ಕಟ್ಟಣೆ

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-6-1"></a>

### ೬.೧ ಇಂಗ್ಲಿಶ್ ಮಾಡುವಂಥದ್ದು ಕನ್ನಡ ಮಾಡಲಾಗದು

ಇಂಗ್ಲಿಶ್ನಲ್ಲಿ ಎಸಕ ಪದಗಳನ್ನು ಯಾವ ಒಟ್ಟು ಸೇರಿಸದೆಯೇ ನೇರ ಹೆಸರು ಪದಗಳಾಗಿ ಬಳಸಬಹುದು — ಇದನ್ನು ಶೂನ್ಯ ಕಟ್ಟಣೆ ಎನ್ನಲಾಗುತ್ತದೆ (ಉದಾ: cover ಎಂಬ ಪದ ಹೊದೆ ಎಂಬ ಎಸಕ ಪದವಾಗಿಯೂ ಮತ್ತು ಹೊದಿಕೆ ಎಂಬ ಹೆಸರು ಪದವಾಗಿಯೂ ಬರುತ್ತದೆ). ಕನ್ನಡದಲ್ಲಿ ಈ ರೀತಿ ಒಟ್ಟಿಲ್ಲದೆ ಬಳಸಲು ಸೀಮಿತ ಅವಕಾಶಗಳಿದ್ದು, ಕೂಗು, ಓದು, ಕೆಮ್ಮು ಮೊದಲಾದ ಕೆಲವು ಎಸಕ ಪದಗಳನ್ನು ಮಾತ್ರ ಹಾಗೆಯೇ ಹೆಸರು ಪದಗಳಾಗಿ ಬಳಸಲು ಬರುತ್ತದೆ. ಹೆಚ್ಚಿನ ಕಡೆಗಳಲ್ಲಿ ಒಟ್ಟನ್ನು ಸೇರಿಸಿ ಕಟ್ಟು ಪದ ಮಾಡಬೇಕಾಗುತ್ತದೆ; ಇದು ಇಂಗ್ಲಿಶ್ ಮತ್ತು ಕನ್ನಡ ನಡುವಿನ ಒಂದು ಮೂಲ ಭಿನ್ನತೆ.

<a id="ch7"></a>

## ಭಾಗ ೭ — ಹೆಸರುಪದದಿಂದ ಹೆಸರುಪದ

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-7-1"></a>

### ೭.೧ ತಜ್ಞ ಒಟ್ಟು -ಅರಿಗ

ತಜ್ಞ ಮಂದಿಯನ್ನು ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ -ಅರಿಗ (-ಗ/-ಇಗ ಒಟ್ಟಿನ ವಿಸ್ತರಣೆ) ಬಳಸಲಾಗುತ್ತದೆ; ಇದು ಇಂಗ್ಲಿಶ್ -ologist, -ist ಒಟ್ಟುಗಳ ಸಾಟಿ. ಒಂದು ಅರಿಮೆ ಕ್ಷೇತ್ರದ ತಜ್ಞನ ಹೆಸರು ಪದ ಕಟ್ಟಲು ಆ ಅರಿಮೆ ಹೆಸರಿನ ಬಳಿಕ -ಅರಿಗ/-ಗ ಸೇರಿಸಲಾಗುತ್ತದೆ — ಉದಾ: ಉಸಿರರಿಮೆ (biology) → ಉಸಿರರಿಮೆಗ (biologist), ಪುರುಳರಿಮೆ (physics) → ಪುರುಳರಿಮೆಗ (physicist). ಹೀಗೆ ಯಾವ ಅರಿಮೆ ಕ್ಷೇತ್ರಕ್ಕೂ ಕನ್ನಡದ ತನ್ನದೇ ತಜ್ಞ ಪದ ಕಟ್ಟಬಹುದು.

<a id="sec-7-2"></a>

### ೭.೨ ಪರ ಒಟ್ಟು -ಒಲವಿಗ

ಒಂದು ವಿಷಯ ಅಥವಾ ನಿಲುವಿಗೆ ಒಲವು ಹೊಂದಿದ ಮಂದಿಯನ್ನು ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ -ಒಲವಿಗ ಬಳಸಲಾಗುತ್ತದೆ; ಇದು ಇಂಗ್ಲಿಶ್ -ist, -phile, enthusiast ಒಟ್ಟುಗಳ ಸಾಟಿ. ಉದಾ: ಕನ್ನಡ + ಒಲವಿಗ = ಕನ್ನಡ ಒಲವಿಗ (Kannada enthusiast/advocate), ಸಮ + ಒಲವಿಗ = ಸಮ ಒಲವಿಗ (socialist). -ಗಾರ ಒಟ್ಟು ವೃತ್ತಿ ಅಥವಾ ಹವ್ಯಾಸ ತಿಳಿಸುತ್ತದೆ; -ಒಲವಿಗ ಒಟ್ಟು ನಿಲುವು ಅಥವಾ ಒಲವು ತಿಳಿಸುತ್ತದೆ — ಎರಡರ ನಡುವಿನ ಈ ಭೇದ ಕನ್ನಡ ಪದ ಕಟ್ಟಿಕೆಯ ಒಂದು ಸೂಕ್ಷ್ಮ ಅಂಶ.

<a id="sec-7-3"></a>

### ೭.೩ ಕಿರಿ ರೂಪ — ಕಿರು-/ಕಿತ್ತ್-

ಕಿರಿದಾದ ಅಥವಾ ಚಿಕ್ಕ ಬಗೆಯ ಒಂದು ಪಾಂಗನ್ನು ಹೆಸರಿಸಲು ಕನ್ನಡದಲ್ಲಿ ಕಿರು- ಎಂಬ ಬೇರನ್ನು ಪದದ ಮುಂದೆ ಸೇರಿಸಲಾಗುತ್ತದೆ; ಇದು ಇಂಗ್ಲಿಶ್ mini- ಒಟ್ಟಿನ ಕನ್ನಡ ಸಾಟಿ. ತೆರೆಯುಲಿ ಮುಂದೆ ಬರುವಾಗ ಕಿರು- ಎಂಬ ರೂಪ ಬರುತ್ತದೆ; ಮುಚ್ಚುಲಿ ಮುಂದೆ ಕಿತ್ತ್- ಎಂಬ ರೂಪ ಬರುತ್ತದೆ. ಉದಾ: ಕಿರುಬೀಡು (mini camp), ಕಿರು ಪದನೆರಕೆ (mini dictionary), ಕಿತ್ತೆಣ್ಣುಕ (minicomputer). ಇದಕ್ಕೆ ತಿರುಗಾಗಿ ದೊಡ್ಡ ಬಗೆ ತಿಳಿಸಲು ಮಿಗಿಲ- ಬೇರನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.

---


<a id="ch8"></a>

## ಭಾಗ ೮ — ಅಳವಿಯ ಮುನ್ನೊಟ್ಟು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-8-1"></a>

### ೮.೧ ಎಣಿಕೆ ಬೇರು ಮುನ್ನೊಟ್ಟುಗಳು

ಅಳವಿಯ ಮುನ್ನೊಟ್ಟುಗಳಲ್ಲಿ ಎಣಿಕೆ ಬೇರುಗಳು ಮುಖ್ಯ ಗುಂಪು ರಚಿಸುತ್ತವೆ. ಒಂದು ಎಂಬ ಹುರುಳಿನ ಒರ್-/ಒರ್- ಬೇರು (ಒರ್ ಮನೆ, ಒರ್ಕಾಲು), ಎರಡು ಎಂಬ ಹುರುಳಿನ ಈರ್-/ಇರ್- ಬೇರು (ಈರ್ ಮಂದಿ, ಇರ್ಟಿ), ಮೂರು ಎಂಬ ಹುರುಳಿನ ಮೂರ್-/ಮುರ್- ಬೇರು, ಮತ್ತು ಅರ್ಧ ಎಂಬ ಹುರುಳಿನ ಅರ್ಧ- ಬೇರು ಬಳಕೆಯಲ್ಲಿವೆ. ತೆರೆಯುಲಿ ಮುಂದೆ ಉದ್ದ ರೂಪ ಮತ್ತು ಮುಚ್ಚುಲಿ ಮುಂದೆ ಗಿಡ್ಡ ರೂಪ ಬರುವುದು ಈ ಬೇರುಗಳ ಸಾಮಾನ್ಯ ನಿಯಮ.

<a id="sec-8-2"></a>

### ೮.೨ ಹಂತ ಮುನ್ನೊಟ್ಟುಗಳು

ಅಳವಿಯ ಮುನ್ನೊಟ್ಟುಗಳ ಎರಡನೇ ಗುಂಪು ಹಂತ (ಡಿಗ್ರಿ) ತಿಳಿಸುವ ಬೇರುಗಳು. ಇಂಗ್ಲಿಶ್ hyper- ಗೆ ಸಾಟಿ ಮಿಗಿಲ- (ಮಿಗಿಲು+ಬ್ಬರ = hyperinflation, ಮಿಗಿಲು+ಗೊಂಡಿ = hyperlink); super- ಗೆ ಸಾಟಿ ಮೇಲ್-/ಮೇ- (ಮೇಲ್ ಮಟ್ಟ); micro-/mini- ಗೆ ಸಾಟಿ ಕಿರು-/ಸಣ್ಣ-; mega- ಗೆ ಸಾಟಿ ದೊಡ್ಡ-. ಹಂತ ತಿಳಿಸುವ ಈ ಬೇರುಗಳು ಇಂಗ್ಲಿಶ್ ಹಂತ ಮುನ್ನೊಟ್ಟುಗಳ ಕನ್ನಡ ಸಾಟಿಗಳಾಗಿ ಬಳಕೆ ಆಗಬಲ್ಲವು.

<a id="ch9"></a>

## ಭಾಗ ೯ — ಇಂಬಿನ ಮುನ್ನೊಟ್ಟು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-9-1"></a>

### ೯.೧ ಇಂಬಿನ ಮುನ್ನೊಟ್ಟು ವ್ಯವಸ್ಥೆ

ಇಂಬಿನ (ಸ್ಥಳ ಸಂಬಂಧ) ಹುರುಳನ್ನು ತಿಳಿಸಲು ಕನ್ನಡದಲ್ಲಿ ಹಲವು ಮುನ್ನೊಟ್ಟು ಬೇರುಗಳು ಬಳಕೆಯಲ್ಲಿವೆ. ಮುನ್- (ಮುಂದೆ), ಹೊರ- (ಹೊರಗೆ), ಒಳ- (ಒಳಗೆ), ನಡು- (ನಡುವೆ) ಮೊದಲಾದ ಬೇರುಗಳನ್ನು ಇಂಗ್ಲಿಶ್ inter-, intra-, extra-, trans-, sub-, super- ಮೊದಲಾದ ಮುನ್ನೊಟ್ಟುಗಳ ಸಾಟಿಯಾಗಿ ಬಳಸಲು ಬರುತ್ತದೆ. ಕನ್ನಡ ಬೇರುಗಳಿಗೆ ಇಂಗ್ಲಿಶ್ ಮುನ್ನೊಟ್ಟುಗಳ ಹಾಗೆ ಒಂದೇ ಅಕ್ಷರ ಜೋಡಣೆ ಇಲ್ಲದಿದ್ದರೂ ಹುರುಳಿನ ಮೂಲಕ ಅದೇ ರೀತಿ ಕೆಲಸ ಮಾಡುತ್ತವೆ.

---


<a id="ch10"></a>

## ಭಾಗ ೧೦ — ಹೊತ್ತಿನ ಮುನ್ನೊಟ್ಟು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-10-1"></a>

### ೧೦.೧ ಹೊತ್ತಿನ ಮುನ್ನೊಟ್ಟು ವ್ಯವಸ್ಥೆ

ಹೊತ್ತಿನ (ಕಾಲ ಸಂಬಂಧ) ಹುರುಳನ್ನು ತಿಳಿಸಲು ಕನ್ನಡದಲ್ಲಿ ಮುನ್- (ಮೊದಲಿನ), ಹಿಂ- (ಹಿಂದಿನ), ಮರ-/ಮತ್ತ- (ಮತ್ತೆ/ಮರಳಿ) ಮೊದಲಾದ ಬೇರುಗಳನ್ನು ಮುನ್ನೊಟ್ಟಿನ ಜಾಗದಲ್ಲಿ ಬಳಸಲಾಗುತ್ತದೆ. ಇಂಗ್ಲಿಶ್ pre-/fore- ಗೆ ಕನ್ನಡ ಸಾಟಿ ಮುನ್-/ಮುಂ- (ಮುಂ+ಗಾಣು = foresee); post- ಗೆ ಸಾಟಿ ಹಿಂ- (ಹಿಂ+ಆಧುನಿಕ = postmodern); re- ಗೆ ಸಾಟಿ ಮರ-/ಮತ್ತ- (ಮರ+ಜನ್ಮ = rebirth); neo- ಗೆ ಸಾಟಿ ಹೊಸ- (ಹೊಸ+ಶಾಸ್ತ್ರೀಯ = neoclassical). ಈ ಬೇರುಗಳ ಸಹಾಯದಿಂದ ಕಾಲ ಸಂಬಂಧ ಪದಗಳನ್ನು ಕನ್ನಡದಲ್ಲೇ ಕಟ್ಟಬಹುದು.

---


<a id="ch11"></a>

## ಭಾಗ ೧೧–೧೨ — ಅಲ್ಲಗಳೆಯುವ ಮುನ್ನೊಟ್ಟು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-11-1"></a>

### ೧೧.೧ ಇಲ್ಲ ಮತ್ತು ಅಲ್ಲ — ಕನ್ನಡದ ವಿಶಿಷ್ಟ ವ್ಯತ್ಯಾಸ

ಕನ್ನಡದಲ್ಲಿ ಅಲ್ಲಗಳೆತ ತಿಳಿಸಲು ಎರಡು ಬೇರೆ ಬೇರೆ ಪದಗಳಿವೆ: ಇಲ್ಲ (ಇರವನ್ನು ಅಲ್ಲಗೆಳೆಯುತ್ತದೆ — ಅಸ್ತಿತ್ವ ನಿಷೇಧ) ಮತ್ತು ಅಲ್ಲ (ಒಂದು ಗುರುತನ್ನು ಅಲ್ಲಗೆಳೆಯುತ್ತದೆ — ಗುರುತು ನಿಷೇಧ). ಉದಾ: "ಅವನು ಮನೆಯಲ್ಲಿ ಇಲ್ಲ" (he is not at home) ಮತ್ತು "ಅವನು ಡಾಕ್ಟರ್ ಅಲ್ಲ" (he is not a doctor). ಈ ಎರಡರ ನಡುವಿನ ವ್ಯತ್ಯಾಸ ಕನ್ನಡ ನುಡಿಯ ವಿಶಿಷ್ಟ ಲಕ್ಷಣ; ಇಂಗ್ಲಿಶ್ not ಈ ಎರಡಕ್ಕೂ ಸಾಟಿಯಾಗುತ್ತದೆ.

<a id="sec-11-2"></a>

### ೧೧.೨ ಅಲ್ಲಗಳೆಯುವ ಮುನ್ನೊಟ್ಟು ಸಮಾನಗಳು

ಇಂಗ್ಲಿಶ್ ಅಲ್ಲಗಳೆಯುವ ಮುನ್ನೊಟ್ಟುಗಳಾದ un-, non-, anti-, dis-, in-/im-/ir-/il- ಗಳಿಗೆ ಸಾಟಿಯಾಗಿ ಕನ್ನಡದಲ್ಲಿ ಹಲವು ರೀತಿ ಬಳಕೆಗಳಿವೆ. ಹೆಸರು ಪದ ಅಥವಾ ಪರಿಚಯ ಪದಕ್ಕೆ ಅಲ್ಲಗಳೆತ ಸೇರಿಸಲು ಅ- ಬೇರು (ಅ + ಒಪ್ಪಿಗೆ = ಅ-ಒಪ್ಪಿಗೆ), ತಪ್ಪು- (wrong/mis-), ಬೇಡ- ಮೊದಲಾದ ಬಳಕೆಗಳು ಇವೆ. ಎಸಕ ಪದಕ್ಕೆ ಅಲ್ಲಗಳೆತ ಸೇರಿಸಲು ಎಸಕ ಪದದ ಅಲ್ಲಗೆಳೆಯುವ ರೂಪಕ್ಕೆ ಇರು ಅಥವಾ ಆಗು ಸೇರಿಸಿ ಬಳಸಲಾಗುತ್ತದೆ.

<a id="sec-11-3"></a>

### ೧೧.೩ ಇತರ ಅಲ್ಲಗಳೆಯುವ ರೀತಿಗಳು

ಕನ್ನಡದಲ್ಲಿ ಅಲ್ಲಗಳೆಯಲು ಬಳಕೆಯಾಗುವ ಹಲವು ಒಟ್ಟುಗಳಿವೆ: ಎಸಕ ಪದಕ್ಕೆ -ಮೆ ಸೇರಿಸಿ ಎಸಕ ನಡೆಯದಿರುವ ಸ್ಥಿತಿ ಹೆಸರಿಸಬಹುದು — ಉದಾ: ಅಂಜು → ಅಂಜಮೆ, ಅರಿ → ಅರಿಯಮೆ, ಸೊಲ್ಲು → ಸೊಲ್ಲಮೆ. ಎಸಕ ಪದದ ಅಲ್ಲಗೆಳೆಯುವ ಜೋಡಿಸುವ ರೂಪ (ಅಂಜದೆ, ಹೇಳದೆ) ಮತ್ತು ಇರು/ಆಗು ಜೋಡಿಸಿ ಕೂಡು ಪದ ಮಾಡುವ ರೀತಿಯೂ ಬಳಕೆಯಲ್ಲಿದೆ. ಇಂಗ್ಲಿಶ್ ಮುನ್ನೊಟ್ಟಿನ ಹಾಗೆ ಒಂದೇ ಬೇರಿನ ಮೂಲಕ ಅಡಕವಾಗಿ ಅಲ್ಲಗಳೆಯಲು ಕನ್ನಡದಲ್ಲಿ ಬರುವುದಿಲ್ಲ — ಇದು ಎರಡು ನುಡಿಗಳ ನಡುವಿನ ಒಂದು ಮೂಲ ಭಿನ್ನತೆ.

---


<a id="ch13"></a>

## ಭಾಗ ೧೩ ಮತ್ತು ೧೭ — ಎಸಕಪದಗಳನ್ನು ಕಟ್ಟುವುದು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-13-1"></a>

### ೧೩.೧ ಒಂದೇ ಉತ್ಪಾದಕ ಎಸಕಪದ ಒಟ್ಟು — -ಇಸು

ಕನ್ನಡದಲ್ಲಿ ಹೊಸ ಎಸಕ ಪದಗಳನ್ನು ಕಟ್ಟಲು ಮುಖ್ಯ ಉತ್ಪಾದಕ ಒಟ್ಟು -ಇಸು. ಹೆಸರು ಪದ, ಪರಿಚಯ ಪದ ಅಥವಾ ಆಣಕ ಪರಿಚಯ ಪದಗಳಿಗೆ -ಇಸು ಸೇರಿಸಿ ಹೊಸ ಎಸಕ ಪದಗಳನ್ನು ಕಟ್ಟಲು ಬರುತ್ತದೆ — ಉದಾ: ಹಂಗು → ಹಂಗಿಸು, ಎತ್ತರ → ಎತ್ತರಿಸು, ಗಮಗಮನೆ → ಗಮಗಮಿಸು. ಇಂಗ್ಲಿಶ್ -ize/-ise, -ify, -en ಒಟ್ಟುಗಳ ಕನ್ನಡ ಸಾಟಿಯಾಗಿ ಬಹಳಷ್ಟು ಕಡೆಗಳಲ್ಲಿ -ಇಸು ಒಟ್ಟನ್ನು ಬಳಸಲು ಬರುತ್ತದೆ.

<a id="sec-13-2"></a>

### ೧೩.೨ ಮೂರು ಭಾಗ ರೂಪಗಳು

ಕನ್ನಡದ ಎಸಕ ಪದಗಳಿಗೆ ನಾಲ್ಕು ಬಗೆಯ ಜೋಡಿಸುವ ರೂಪಗಳಿದ್ದು, ಕೂಡು ಪದ ಮಾಡಲು ಮೊದಲ ಎರಡನ್ನು (ಮೊದಲೆಸಕದ ರೂಪ ಮತ್ತು ಮಾರಸಕದ ರೂಪ) ಬಳಸಲಾಗುತ್ತದೆ. ಉದಾ: ಬರು → ಬಂದು, ಬರಲು; ತಿನ್ನು → ತಿಂದು, ತಿನ್ನಲು — ಈ ಎರಡೂ ರೂಪಗಳು ಕೂಡು ಪದ ಕಟ್ಟಲು ಬಳಕೆ ಆಗುತ್ತವೆ. ಮೊದಲನೇ ರೂಪ ಒಂದು ಎಸಕ ನಡೆದ ಮೇಲೆ ಮತ್ತೊಂದು ನಡೆಯಿತೆಂಬ ಅರ್ಥ ಕೊಡುತ್ತದೆ; ಎರಡನೇ ರೂಪ ಗುರಿ ಅಥವಾ ಕಾರಣ ಅರ್ಥ ಕೊಡುತ್ತದೆ.

---


<a id="ch14"></a>

## ಭಾಗ ೧೪–೧೬ — ಜೋಡುಪದ ಮತ್ತು ನಿಂತ ಹೆಸರುಪದ

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-14-1"></a>

### ೧೪.೧ ಜೋಡು ಹೆಸರುಪದ

ಕನ್ನಡದಲ್ಲಿ ಎರಡು ಹೆಸರು ಪದಗಳನ್ನು, ಒಂದು ಎಸಕ ಪದ ಮತ್ತು ಒಂದು ಹೆಸರು ಪದವನ್ನು, ಅಥವಾ ಒಂದು ಪರಿಚಯ ಪದ ಮತ್ತು ಒಂದು ಹೆಸರು ಪದವನ್ನು ಸೇರಿಸಿ ಜೋಡು ಹೆಸರು ಪದಗಳನ್ನು ಕಟ್ಟಬಹುದು. ಉದಾ: ಮನೆ + ಕೆಲಸ = ಮನೆಕೆಲಸ; ಸುರಿ + ಮಳೆ = ಸುರಿಮಳೆ; ಹೆಬ್ + ಬೆರಳು = ಹೆಬ್ಬೆರಳು; ಕಣ್ + ಸನ್ನೆ = ಕಣ್ಣಸನ್ನೆ. ಜೋಡು ಹೆಸರು ಪದದ ಮೊದಲ ಪದ ಎರಡನೇ ಪದ ಗುರುತಿಸುವ ಪಾಂಗಿನ ಪರಿಚಯ ತಿಳಿಸುತ್ತದೆ.

<a id="sec-14-2"></a>

### ೧೪.೨ ಜೋಡು ಎಸಕಪದ

ಕನ್ನಡದಲ್ಲಿ ಎಸಕ ಪದದ ಜೋಡಿಸುವ ರೂಪಕ್ಕೆ ಇನ್ನೊಂದು ಎಸಕ ಪದವನ್ನು ಸೇರಿಸಿ ಕೂಡು ಪದ ಮಾಡಲಾಗುತ್ತದೆ. ಉದಾ: ಹೇಳು + ಕೊಡು = ಹೇಳಿಕೊಡು; ಕೈ + ಗೂಡು = ಕೈಗೂಡು; ಮರೆ + ಬಿಡು = ಮರೆತುಬಿಡು; ಹೋಗ + ಲಾಡಿಸು = ಹೋಗಲಾಡಿಸು. ಮೊದಲ ಎಸಕ ಪದ ಜೋಡಿಸುವ ರೂಪ ಪಡೆದ ಮೇಲೆ ಅದರ ಬಳಿಕ ಎರಡನೇ ಎಸಕ ಪದ ಬರುತ್ತದೆ; ಹೀಗೆ ಕಟ್ಟಿದ ಕೂಡು ಪದ ಕನ್ನಡ ಎಸಕ ಪದ ಕಟ್ಟಿಕೆಯ ಮುಖ್ಯ ಹಮ್ಮುಗೆ.

<a id="sec-14-3"></a>

### ೧೪.೩ ಜೋಡಣೆಯ ಮುಖ್ಯ ನಿಯಮ

ಜೋಡು ಪದ ಮತ್ತು ಕೂಡು ಪದ ಕಟ್ಟುವಲ್ಲಿ ಮುಖ್ಯ ನಿಯಮ: ಮೊದಲ ಪದ ಯಾವ ಪದ ವರ್ಗಕ್ಕೆ ಸೇರಿದ್ದರೂ ಅದು ಬೇರಿನ ರೂಪದಲ್ಲಿ ಅಥವಾ ಜೋಡಿಸುವ ರೂಪದಲ್ಲಿ ಬರಬೇಕು. ಉದಾ: ಹಿಂದೆ + ಸರಿ = ಹಿಂಜರಿ; ಮುಂದೆ + ನಡೆ = ಮುನ್ನಡೆ; ಮೇಲಕ್ಕೆ + ಏರು = ಮೇಲೇರು. ಒಟ್ಟು ಸೇರಿಸಿ ಕಟ್ಟು ಪದ ಮಾಡಲು ಬಾರದ ಕಡೆ ಕೂಡು ಪದ ಹೊಲವನ್ನು ಬಳಸುವುದು ಕನ್ನಡದ ಹೆಚ್ಚು ನೈಸರ್ಗಿಕ ಒಲವು.

<a id="sec-15-1"></a>

### ೧೫.೧ ನಿಂತ ಇಂಗ್ಲಿಶ್ ಹೆಸರುಪದ — ತೀರ್ಮಾನ ಕ್ರಮ

ಇಂಗ್ಲಿಶ್ ಪದಗಳು ಕನ್ನಡ ಬಳಕೆಯಲ್ಲಿ ನಿಂತಿದ್ದಲ್ಲಿ ಅವಕ್ಕೆ ಕನ್ನಡ ಸಾಟಿ ಕಟ್ಟಲು ನಿರ್ದಿಷ್ಟ ಕ್ರಮ ಅನುಸರಿಸಬಹುದು. ಮೊದಲು ಆ ಇಂಗ್ಲಿಶ್ ಪದದ ಹುರುಳು ಏನೆಂಬುದನ್ನು ನಿಖರವಾಗಿ ಅರಿಯಬೇಕು; ಆಮೇಲೆ ಅದೇ ಹುರುಳಿನ ಕನ್ನಡ ಪದ ಬಳಕೆಯಲ್ಲಿದೆಯೇ ನೋಡಬೇಕು; ಇಲ್ಲದಿದ್ದಲ್ಲಿ ಕಟ್ಟು ಪದ ಅಥವಾ ಜೋಡು ಪದ ಮಾಡಬೇಕು. ಈ ಕ್ರಮ ಬಳಸಿ ಯಾವ ಇಂಗ್ಲಿಶ್ ಹೆಸರು ಪದಕ್ಕೂ ಕನ್ನಡ ಸಾಟಿ ಕಟ್ಟಬಹುದು.

## ಭಾಗ ೧೮ — ನವ-ಶಾಸ್ತ್ರೀಯ ಬೇರುಗಳು

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="sec-18-1"></a>

### ೧೮.೧ ಬೇರು ಸಮಾನಗಳು

ಗ್ರೀಕ್ ಅಥವಾ ಲ್ಯಾಟಿನ್ ಬೇರುಗಳನ್ನು ಬಳಸಿ ಇಂಗ್ಲಿಶ್ನಲ್ಲಿ ಕಟ್ಟಿದ ಹೊಸ-ಶಾಸ್ತ್ರೀಯ ಜೋಡು ಪದಗಳಿಗೆ (neo-classical compounds) ಸಾಟಿ ಕನ್ನಡ ಪದ ಕಟ್ಟಲು ಆ ಎರವಲು ಬೇರುಗಳ ಹುರುಳನ್ನು ಕನ್ನಡ ಪದ ಅಥವಾ ಬೇರಿನಿಂದ ಪಡೆಯಲು ಬರುತ್ತದೆ. ಉದಾ: astro- (star/space) = ಬಾನ್- (ಬಾನ್ + ಅರಿಮೆ = astrophysics); bio- (life) = ಉಸಿರ- (ಉಸಿರರಿಮೆ = biology); geo- (earth) = ನೆಲ- (ನೆಲ ಅರಿಮೆ = geology). ಹೀಗೆ ಕನ್ನಡ ಬೇರುಗಳ ಮೂಲಕ ಯಾವ ಶಾಸ್ತ್ರ ಕ್ಷೇತ್ರಕ್ಕೂ ಕನ್ನಡ ಪದ ಕಟ್ಟಬಹುದು.

<a id="sec-18-2"></a>

### ೧೮.೨ ಒಟ್ಟು ಸಮಾನಗಳು

ಇಂಗ್ಲಿಶ್ ಹೊಸ-ಶಾಸ್ತ್ರೀಯ ಜೋಡು ಪದಗಳಲ್ಲಿ ಬಳಕೆ ಆಗುವ -logy, -graphy, -metry, -scope ಮೊದಲಾದ ಒಟ್ಟುಗಳಿಗೆ ಸಾಟಿಯಾಗಿ ಕನ್ನಡ ಒಟ್ಟು ಅಥವಾ ಬೇರನ್ನು ಬಳಸಲು ಬರುತ್ತದೆ. ಉದಾ: -logy (study of) = -ಅರಿಮೆ (biology = ಉಸಿರರಿಮೆ); -metry (measurement) = -ಅಳತೆ (geometry = ನೆಲ ಅಳತೆ); -graphy (writing/mapping) = -ಬರಹ (geography = ನೆಲ ಬರಹ). ಈ ಸಾಟಿಗಳ ಸಹಾಯದಿಂದ ಅರಿಮೆ ಕ್ಷೇತ್ರಗಳ ಪದಗಳನ್ನು ಸಂಸ್ಕೃತ ಅಥವಾ ಇಂಗ್ಲಿಶ್ ಒಟ್ಟು ಆಶ್ರಯಿಸದೆ ಕನ್ನಡದಲ್ಲೇ ಕಟ್ಟಬಹುದು.

## ಭಾಗ ೧೯–೫೨ — ಅನ್ವಯ — ಇಂಗ್ಲಿಶ್ ಒಟ್ಟುಗಳ ವಿಶ್ಲೇಷಣೆ

[↑ ಒಳಪಿಡಿಗೆ ಹಿಂತಿರುಗಿ](#toc)

---


<a id="ch19-28"></a>

### ಭಾಗ ೧೯–೨೮ — ಹೆಸರುಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳು


<a id="sec-19-1"></a>

#### ೧೯.೧ ಎಸಗುಗ ಹಿನ್ನೊಟ್ಟುಗಳು — -er/-or, -ist, -eer, -ian

ಇಂಗ್ಲಿಶ್ ಹೆಸರು ಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳ ಪ್ರಯೋಗ ಭಾಗ ೧೯–೨೮ ರ ಮುಖ್ಯ ವಿಷಯ. ಎಸಗುಗ ಹಿನ್ನೊಟ್ಟುಗಳಾದ -er/-or, -ist, -eer, -ian ಇಂಗ್ಲಿಶ್ನಲ್ಲಿ ಮಂದಿ ಹೆಸರಿಸಲು ಬಳಕೆ ಆಗುತ್ತವೆ. ಇವಕ್ಕೆ ಸಾಟಿಯಾಗಿ ಕನ್ನಡದಲ್ಲಿ -ಗ/-ಇಗ ಅಥವಾ -ಗಾರ ಒಟ್ಟನ್ನು ಬಳಸಲು ಬರುತ್ತದೆ — ಉದಾ: reader = ಓದುಗ; gardener = ತೋಟಗಾರ; researcher = ಅರಕೆಗಾರ; musician = ಸಂಗೀತಿಗ. ಎಸಕ ಪದ ಸಿಗದ ಕಡೆ ಹೆಸರು ಪದದ ಬಳಿಕ -ಗ/-ಗಾರ ಸೇರಿಸಲಾಗುತ್ತದೆ.

<a id="sec-19-2"></a>

#### ೧೯.೨ ಅಮೂರ್ತ ಗುಣ ಹಿನ್ನೊಟ್ಟುಗಳು — -ness, -ity, -ism, -dom, -hood, -ship

ಇಂಗ್ಲಿಶ್ ಅಮೂರ್ತ ಗುಣ ಹಿನ್ನೊಟ್ಟುಗಳಾದ -ness, -ity, -ism, -dom, -hood, -ship ಗಳು ಪರಿಚಯ ಪದ ಅಥವಾ ಹೆಸರು ಪದದಿಂದ ಅಮೂರ್ತ ಹೆಸರು ಪದ ಕಟ್ಟುತ್ತವೆ. ಇವಕ್ಕೆ ಸಾಟಿಯಾಗಿ ಕನ್ನಡದಲ್ಲಿ -ತನ ಒಟ್ಟನ್ನು ಬಳಸಲಾಗುತ್ತದೆ — ಉದಾ: boldness = ದಿಟ್ಟತನ; brotherhood = ಅಣ್ಣ-ತಮ್ಮತನ; friendship = ಗೆಳೆತನ. -ism ಗೆ ಸಾಟಿ ಸಂದರ್ಭ ಅನುಸಾರ ಒಲವು (ism as belief/tendency) ಅಥವಾ ವಾದ ಎಂಬ ಕನ್ನಡ ಪದ ಬಳಕೆ ಆಗಬಹುದು.

<a id="sec-19-3"></a>

#### ೧೯.೩ ಎಸಕ ಮತ್ತು ಕ್ರಿಯೆ ಹಿನ್ನೊಟ್ಟುಗಳು — -tion/-sion, -ment, -ance/-ence, -ure, -age

ಇಂಗ್ಲಿಶ್ ಎಸಕ ಮತ್ತು ಕ್ರಿಯೆ ತಿಳಿಸುವ ಹಿನ್ನೊಟ್ಟುಗಳಾದ -tion/-sion, -ment, -ance/-ence, -ure, -age ಗಳು ಎಸಕ ಪದದಿಂದ ಅಮೂರ್ತ ಹೆಸರು ಪದ ಕಟ್ಟುತ್ತವೆ. ಇವಕ್ಕೆ ಸಾಟಿಯಾಗಿ ಕನ್ನಡದಲ್ಲಿ ಮುಖ್ಯವಾಗಿ -ಇಕೆ/-ಕೆ ಅಥವಾ -ತ ಒಟ್ಟನ್ನು ಬಳಸಲಾಗುತ್ತದೆ — ಉದಾ: migration = ವಲಸೆ; replacement = ಬದಲಾವಣೆ; attendance = ಹಾಜರಾತಿ. ಇಂಗ್ಲಿಶ್ ಎಸಕ ಪದಕ್ಕೆ ಸಾಟಿ ಕನ್ನಡ ಎಸಕ ಪದ ಸಿಗದಿದ್ದಲ್ಲಿ ಕೂಡು ಪದ ರೂಪ ಬಳಸಿ ಹೆಸರಿಸಬಹುದು.

<a id="sec-19-4"></a>

#### ೧೯.೪ ಫಲ ಮತ್ತು ಉತ್ಪತ್ತಿ ಹಿನ್ನೊಟ್ಟುಗಳು — -ure, -age (ಫಲ)

ಇಂಗ್ಲಿಶ್ -ure (ಉದಾ: closure, seizure) ಮತ್ತು -age (ಉದಾ: wreckage, leakage) ಒಟ್ಟುಗಳು ಎಸಕದ ಮೂರ್ತ ಫಲ ಅಥವಾ ದೊರೆತ ಪಾಂಗ ಹೆಸರಿಸಲು ಬಳಕೆ ಆಗುತ್ತವೆ. ಕನ್ನಡದಲ್ಲಿ ಇಂತಹ ಪದಗಳಿಗೆ ಸಾಟಿ ಕಟ್ಟಲು -ಗೆ/-ಇಗೆ ಅಥವಾ -ತೆ ಒಟ್ಟನ್ನು ಬಳಸಲಾಗುತ್ತದೆ — ಉದಾ: closure = ಮುಚ್ಚಿಗೆ; seizure = ಹಿಡಿತ; leakage = ಸೋರಿಗೆ. ಅಮೂರ್ತ ಯಸಕ ತಿಳಿಸುವ -ಇಕೆ/-ಕೆ, -ತ ಒಟ್ಟುಗಳಿಗಿಂತ ಭಿನ್ನವಾಗಿ ಇವು ಮೂರ್ತ ಫಲ ಹೆಸರಿಸುತ್ತವೆ.

<a id="ch29-36"></a>

### ಭಾಗ ೨೯–೩೬ — ಎಸಕಪದ ಮತ್ತು ಪರಿಚೆಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳು


<a id="sec-29-1"></a>

#### ೨೯.೧ ಎಸಕಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳು — -ize/-ise, -ify, -en

ಇಂಗ್ಲಿಶ್ ಎಸಕ ಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳಾದ -ize/-ise, -ify, -en ಗಳು ಹೆಸರು ಪದ ಅಥವಾ ಪರಿಚಯ ಪದದಿಂದ ಎಸಕ ಪದ ಕಟ್ಟಲು ಬಳಕೆ ಆಗುತ್ತವೆ. ಕನ್ನಡದಲ್ಲಿ ಇವಕ್ಕೆ ಸಾಟಿಯಾಗಿ ಮುಖ್ಯವಾಗಿ -ಇಸು ಒಟ್ಟನ್ನು ಬಳಸಲಾಗುತ್ತದೆ — ಉದಾ: broaden = ಅಗಲಿಸು; loosen = ಸಡಿಲಿಸು; falsify = ಸುಳ್ಳುಮಾಡು; awaken = ಎಚ್ಚರಿಸು. ಒಟ್ಟು ಸೇರಿಸಲು ಆಗದ ಕಡೆ ಕೂಡು ಪದ ರೂಪ ಬಳಸಬೇಕಾಗುತ್ತದೆ — ಉದಾ: shorten = ಮೊಟಕುಗೊಳಿಸು.

<a id="sec-29-2"></a>

#### ೨೯.೨ ಪರಿಚೆಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳು — -ful, -less, -ous, -ive, -ible/-able, -al, -ic

ಇಂಗ್ಲಿಶ್ ಪರಿಚಯ ಪದ ಕಟ್ಟುವ ಹಿನ್ನೊಟ್ಟುಗಳಾದ -ful, -less, -ous, -ive, -ible/-able, -al, -ic ಗಳಿಗೆ ಸಾಟಿ ಕನ್ನಡ ಪದ ಕಟ್ಟಲು ಹಲವು ಹಮ್ಮುಗೆಗಳಿವೆ. ಇರವು ತಿಳಿಸಲು ಉಳ್ಳ/ಇರುವ, ಅಭಾವ ತಿಳಿಸಲು ಇಲ್ಲದ, ಸಾಧ್ಯತೆ ತಿಳಿಸಲು ಬಲ್ಲ/ಬರುವ, ಸಂಬಂಧ ತಿಳಿಸಲು ಪತ್ತುಗೆ ರೂಪ (-ದ/-ಯ) ಬಳಕೆ ಆಗುತ್ತದೆ — ಉದಾ: hopeful = ನಿರೀಕ್ಷೆ ಉಳ್ಳ; useless = ಉಪಯೋಗ ಇಲ್ಲದ; readable = ಓದಬಲ್ಲ. ಕನ್ನಡದಲ್ಲಿ ಒಂದೇ ಒಟ್ಟಿನಲ್ಲಿ ಇವನ್ನೆಲ್ಲ ಅಡಕಗೊಳಿಸಲು ಆಗದಿರುವುದು ಒಂದು ಸ್ಥಳೀಯ ಅಂತರ.

<a id="ch37-52"></a>

### ಭಾಗ ೩೭–೫೨ — ಇಂಗ್ಲಿಶ್ ಮುನ್ನೊಟ್ಟುಗಳ ವಿಶ್ಲೇಷಣೆ


<a id="sec-37-1"></a>

#### ೩೭.೧ ಅಳವಿ/ಹಂತ ಮುನ್ನೊಟ್ಟುಗಳು — hyper-, mega-, micro-, mini-, super-

ಇಂಗ್ಲಿಶ್ ಅಳವಿ ಮತ್ತು ಹಂತ ತಿಳಿಸುವ ಮುನ್ನೊಟ್ಟುಗಳಾದ hyper-, mega-, micro-, mini-, super- ಗಳಿಗೆ ಸಾಟಿ ಕನ್ನಡ ಬೇರುಗಳು ಬಳಕೆ ಆಗಬಲ್ಲವು. hyper- ಗೆ ಮಿಗಿಲ- (ಮಿಗಿಲು+ಬ್ಬರ = hyperinflation; ಮಿಗಿಲು+ಗೊಂಡಿ = hyperlink); mini- ಗೆ ಕಿರು- (ಕಿರು+ಬೀಡು = mini camp); micro- ಗೆ ಸಣ್ಣ-; mega-/super- ಗೆ ದೊಡ್ಡ-/ಮಿಗಿಲ- ಬಳಸಲಾಗುತ್ತದೆ. ತೆರೆಯುಲಿ ಮುಂದೆ ಕಿರು- ಮತ್ತು ಮುಚ್ಚುಲಿ ಮುಂದೆ ಕಿತ್ತ್- ಎಂಬ ಮಾರ್ಪಾಡು ರೂಪ ಬರುತ್ತದೆ.

<a id="sec-37-2"></a>

#### ೩೭.೨ ಇಂಬಿನ ಮುನ್ನೊಟ್ಟುಗಳು — inter-, intra-, extra-, trans-, sub-, super-

ಇಂಗ್ಲಿಶ್ ಇಂಬಿನ (ಸ್ಥಳ) ಮುನ್ನೊಟ್ಟುಗಳಾದ inter-, intra-, extra-, trans-, sub-, super- ಗಳಿಗೆ ಸಾಟಿ ಕನ್ನಡ ಬೇರುಗಳನ್ನು ಬಳಸಬಹುದು. inter- ಗೆ ನಡು- (ನಡು+ನಾಡು = interstate); intra- ಗೆ ಒಳ- (ಒಳ+ನಾಡು = intrastate); trans- ಗೆ ದಾಟ-/ಅಡ್ಡ-; sub- ಗೆ ಕೆಳ-; extra- ಗೆ ಹೊರ- (ಹೊರ+ನಾಡು = extraterritorial). ಕನ್ನಡ ಬೇರುಗಳು ಇಂಗ್ಲಿಶ್ ಮುನ್ನೊಟ್ಟುಗಳ ಹಾಗೆ ನೇರ ಪ್ರಿಫಿಕ್ಸ್ ಆಗಿ ಅಲ್ಲ, ಆದರೆ ಸ್ವತಂತ್ರ ಬೇರಾಗಿ ಸೇರಿ ಜೋಡು ಪದ ಕಟ್ಟುತ್ತವೆ.

<a id="sec-37-3"></a>

#### ೩೭.೩ ಹೊತ್ತಿನ ಮುನ್ನೊಟ್ಟುಗಳು — pre-, post-, re-, ex-, neo-, paleo-

ಇಂಗ್ಲಿಶ್ ಹೊತ್ತಿನ ಮುನ್ನೊಟ್ಟುಗಳಾದ pre-, post-, re-, ex-, neo-, paleo- ಗಳಿಗೆ ಸಾಟಿ ಕನ್ನಡ ಬೇರುಗಳನ್ನು ಬಳಸಬಹುದು. pre-/fore- ಗೆ ಮುನ್-/ಮುಂ- (ಮುಂ+ಗಾಣು = foresee); post- ಗೆ ಹಿಂ- (ಹಿಂ+ಆಧುನಿಕ = postmodern); re- ಗೆ ಮರ-/ಮತ್ತ- (ಮರ+ಜನ್ಮ = rebirth); ex- ಗೆ ಮಾಜಿ- (ಮಾಜಿ ಮುಖ್ಯಮಂತ್ರಿ = ex-CM); neo- ಗೆ ಹೊಸ- (ಹೊಸ+ಶಾಸ್ತ್ರೀಯ = neoclassical). ಈ ಬೇರುಗಳು ಕಾಲ ಸಂಬಂಧ ಪದಗಳಿಗೆ ಕನ್ನಡ ನೆಲ ಒದಗಿಸುತ್ತವೆ.
