# ಸೊಲ್ಲು ಕಟ್ಟುವ ಕೂಟ — Word Coining Sessions

*Coining native Dravidian Kannada words using Dr. D.N. Shankara Bhat's Ellara methodology.*

These sessions work through English, Sanskrit-loan, and borrowed terms — replacing them with native Dravidian coinages following the approach DNS Bhat used to produce words like ಎಲ್ಲರಾಳ್ವಿಕೆ (democracy), ಉಸಿರಿಯರಿಮೆ (biology), and ಗೆಂಟುತೋರ್ಪುಕ (telescope).
