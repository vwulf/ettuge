# ಸೊಲ್ಲು ಕಟ್ಟುವ ಕೂಟ — Word Coining Sessions

*Coining native Dravidian Kannada words using Dr. D.N. Shankara Bhat's Ellara methodology.*

These sessions work through English technical and everyday terms, replacing Sanskrit loans and direct borrowings with native Dravidian coinages — the approach DNS Bhat used to produce words like ಎಲ್ಲರಾಳ್ವಿಕೆ (democracy), ಉಸಿರಿಯರಿಮೆ (biology), and ಗೆಂಟುತೋರ್ಪುಕ (telescope).

## Session 1 — Paragraph Rewrite Exercise

*Source: Kannada newspaper column asking readers to coin words for nodal, squad, template.*

### Original Paragraph

ಈ ವಾರದ ಮೂರು ಪದಗಳು:
1. Nodal: ನೋಡಲ್ ಎಂದು ಕನ್ನಡದಲ್ಲಿ ಬರೆದರೂ, ಹಳಗನ್ನಡದಂತೆ ಧ್ವನಿಸಿದರೂ, ಕನ್ನಡವಲ್ಲ! ಆದ್ದರಿಂದ ಹೊಸ ಪದ ಬೇಕಿದೆ.
2. Squad: ನಿರ್ದಿಷ್ಟ ಕಾರ್ಯ, ತನಿಖೆ ಅಥವಾ ಕಾರ್ಯಾಚರಣೆಗಾಗಿ ನೇಮಕಗೊಂಡ ವಿಶೇಷ ತಂಡ.
3. Template: ಮಾದರಿಯೂ ಮಾರ್ಗದರ್ಶಿಯೂ ಹೌದಾದರೂ ಅವೇ ಪದಗಳು ಸಮರ್ಥವೆನಿಸವು. ಹೊಸದೊಂದು ಕನ್ನಡ ಪದ ಬೇಕು.
ಓದುಗರು ಪ್ರತಿಯೊಂದು ಅಂಗ್ಲ ಪದಕ್ಕೆ ತಲಾ ಒಂದರಂತೆ ಕನ್ನಡ ಪದ ತಿಳಿಸಬೇಕು. ಶನಿವಾರ ಸಂಜೆ ೬ ಗಂಟೆಯೊಳಗೆ.
*(ಲೇಖಕರು ಅಮೆರಿಕದಲ್ಲಿ ಸಾಫ್ಟ್‌ವೇರ್ ಇಂಜಿನಿಯರ್ ಮತ್ತು ಅಂಕಣ ಬರಹಗಾರರು)*

### Native Rewrite

ಈ ಏಳ್ನಾಳಿನ ಮೂರು ಸೊಲ್ಲುಗಳು:
1. Nodal: ನೋಡಲ್ ಎಂದು ಕನ್ನಡದಲ್ಲಿ ಬರೆದರೂ, ಹಳಗನ್ನಡದಂತೆ ಉಲಿದರೂ, ಕನ್ನಡವಲ್ಲ! ಆದ್ದರಿಂದ ಹೊಸ ಸೊಲ್ಲು ಬೇಕಿದೆ.
2. Squad: ಬೇರ್ಗೊಂಡ ಕೆಲಸ, ಆರಾಯ್ಕೆ ಇಲ್ಲವೇ ನಡೆಸಿಕೆಗಾಗಿ ಆಯ್ಕೆಗೊಂಡ ಹೆಗ್ಗತಂಡ.
3. Template: ಅಚ್ಚೂ ಅರಿವೀಯುಗನೂ ಹೌದಾದರೂ ಅವೇ ಸೊಲ್ಲುಗಳು ಅಳವುಳ್ಳವೆನಿಸವು. ಹೊಸದೊಂದು ಕನ್ನಡ ಸೊಲ್ಲು ಬೇಕು.
ಓದುಗರು ಒಂದೊಂದು ಇಂಗ್ಲಿಶ್ ಸೊಲ್ಲಿಗೆ ಒಂದೊಂದರಂತೆ ಕನ್ನಡ ಸೊಲ್ಲು ತಿಳಿಸಬೇಕು. ಮುಂದಿನ ಕಾದೇರನಾಳು ಬೈಗು ೬ ಆಗೋಹೊತ್ತಿನೊಳಗೆ.
*(ಬರಹಗಾರರು ಅಮೆರಿಕದಲ್ಲಿ ಮಿನ್ನವಿರು ಬಿಣಿಗೆಯರಿಗ ಮತ್ತು ಅಂಕಣ ಬರಹಗಾರರು)*

## Session 2 — Day Names (ನಾಳು System)

*All day names built on -ನಾಳು (nALu = day), the same root as ಏಳ್ನಾಳು (week = seven-days).*

All roots are Dravidian (ಹೆರೆ, ನಲ್, ಚೆನ್ನ, ಜಾಣ್, ಓಜ, ಕಲಿ, ಬೆಳ್ಳಿ, ಕಾದೇರ, ನೇಸರ, ನಾಳು ✅).

### ಕಾದೇರ — Saturn Etymology

**ಕಾದೇರ** (kAdEra) is the native name for Saturn. The preferred reading:

- ಕಾದು (*kAdu*) = to wait — Dravidian ✅
- ಏರ (*Era*) = one who rises, from ಏರು (to climb) — Dravidian ✅
- Meaning: "the slow-rising waiter" — Saturn spends 2+ years per zodiac sign, the slowest-moving visible planet. A precise astronomical description in purely native terms.

|-------------|--------------|-------------|----------------|
| Moon | ಸೋಮ | ಹೆರೆ | moon |

| Mercury | ಬುಧ | ಜಾಣ್ | wise/clever |

## Session 3 — Electronic/Computing Terms & ಮಿನ್ನು- Convention

### ಮಿನ್ನು- (minnu-): the Electronic/Digital Prefix

**ಮಿನ್ನು** (minnu) = electricity, flash, lightning — Dravidian. Used as the productive prefix for anything in the electronic/digital domain, parallel to **ಗೆಂಟು-** (tele-/distant):

| Domain prefix | Meaning | Example |
|--------------|---------|---------|
| ಮಿನ್ನು- | electronic/digital | ಮಿನ್ನುಕ (computer), ಮಿನ್ನವಿರು (software) |
| ಗೆಂಟು- | tele-/distant | ಗೆಂಟುತೋರ್ಪುಕ (telescope), ಗೆಂಟುಲಿಕ (telephone) |
| ಹೆಗ್ಗ- | great/distinguished | ಹೆಗ್ಗತಂಡ (special squad) |

### Computing Coinages

| English | Sanskrit/borrowed | Native | Eke | Notes |
|---------|------------------|--------|-----|-------|
| computer | ಗಣಕ (Sanskrit) | ಮಿನ್ನುಕ | minnuka | ಮಿನ್ನು + -ಕ (instrument) |
| software | ತಂತ್ರಾಂಶ (Sanskrit) | ಮಿನ್ನವಿರು | minnaviru | ಮಿನ್ನು + ಅವಿರು (thread/filament) |

**Why not ಎಣ್ಣುಕ (counting-device)?** It's native but generic — doesn't distinguish electronic computer from an abacus. ಮಿನ್ನು- correctly scopes the electronic domain.

## Formation Patterns

| Pattern | Suffix/Prefix | Example |
|---------|--------------|---------|
| Instrument noun | -ಕ | ಅಚ್ಚು → ಅಚ್ಚುಕ (template), ಮಿನ್ನು → ಮಿನ್ನುಕ (computer) |
| Genitive adjective | -ಇನ | ಗಂಟು → ಗಂಟಿನ (nodal) |
| Diminutive prefix | ಕಿರು- | ಕಿರು + ತಂಡ → ಕಿರುತಂಡ (squad) |
| Electronic prefix | ಮಿನ್ನು- | ಮಿನ್ನು + ಅವಿರು → ಮಿನ್ನವಿರು (software) |
| Compound + suffix | [root] + -ಕೆ | ಆರಾಯ್ → ಆರಾಯ್ಕೆ (investigation) |
| Calendar suffix | [body] + ನಾಳು | ಹೆರೆ + ನಾಳು → ಹೆರೆನಾಳು (Monday) |

## Key Principles

1. **Step 1: check if native already exists.** If ಸೊಲ್ಲು, ಕೆಲಸ, ತಂಡ cover the meaning, use them — don't reinvent.
2. **Pre-vetted coinages take precedence** over fresh coinages, even when the fresh coinage is morphologically sound.
3. **Translate the meaning, not the word form.** "Software" → ಮಿನ್ನವಿರು (electronic-thread), not *ಮಿನ್ನು-ಮೃದು* (electronic-soft).
4. **Prefer suffix-derived over compounds over loanwords.** Compact > descriptive > borrowed.
5. **Retroflex ಡ/ಣ/ಳ in a root** is a signal the root is likely Dravidian.
6. **Telugu/Tamil cognates confirm Dravidian origin** only when the correspondence is regular — parallel Sanskrit borrowing is also possible (verify with Wiktionary etymology field).
7. **Avoid mahapranas in native coinages** — aspirated consonants don't occur in native Dravidian Kannada; they entered via Sanskrit loans.
