# Eke — Ellara Kannada Romanisation

**Eke** (ಏಕೆ) is the romanisation system for Kannada used in the ettuge project. It enables precise, reversible romanisation of Kannada script using only ASCII characters.

## Key principles

- **Case sensitivity** — uppercase marks retroflex, long, or special: `T D N L S A I U E O H`
- **No standalone M** — anusvara ಂ always assimilates to the following consonant
- **`r` is always ರ** — never use `R` (= rare archaic ಱ) for the common r sound
- **Diphthongs as sequences** — ಐ = `ay`, ಔ = `av` (not `ai`, `au`)
- **Two modes** — full Eke preserves aspirates; Eke(ek) drops them for native word coining
