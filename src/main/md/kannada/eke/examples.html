Two Eke modes: When romanising existing Kannada texts, aspirates are preserved — ಭ → bh, ಧ → dh, ಖ → kh etc. When coining new native words (Eke(ek) mode), aspirates are dropped since native Dravidian roots have no mahaprāṇa consonants. The examples on this page use the romanisation mode (aspirates preserved).

Optional h in classical/Sanskrit texts: Where a text contains Sanskrit loanwords, the optional aspiration marker h is shown in parentheses — e.g. kas(h)Ta, vid(h)i — indicating that Ellara Kannada speakers may drop the aspiration in natural speech.

Brāhmī script: characters are in Asokan Brāhmī (Unicode block 11000–1107F). Note that Brāhmī, like Devanāgarī, does not distinguish short-e/long-E or short-o/long-O.

ಹಲ್ಮಿಡಿ ಶಾಸನ — Halmiḍi Inscription
The earliest known Kannada inscription (c. 450 CE), originally in Kadamba script. Transcribed here via modern Kannada script (from Wikipedia's Kannada-script rendering) and romanised in Eke. The original Kadamba script is related to both modern Kannada and Asokan/Tamil Brāhmī. Sanskrit loanwords are heavily present — aspirates fully preserved in Eke.
