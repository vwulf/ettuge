# Native Kannada Word for 'Democracy' — DNS Bhat Style

## Result

**English**: democracy
**Kannada**: ಮಂದಿಯಾಳ್ವಿಕೆ
**Eke**: mandiyALvike
**Breakdown**: ಮಂದಿ (people) + ಆಳ್ವಿಕೆ (rule/governance)
**Pattern**: Like ಉಸಿರಿಯರಿಮೆ (biology = life + study-of) — semantic compound of a native noun + a native governance suffix

### Why Not These Alternatives?

| Candidate | Problem |
|-----------|---------|
| ಜನತಾಡಳಿತ | ಜನತಾ is Sanskrit-derived; rejected |
| ಪ್ರಜಾಪ್ರಭುತ್ವ | Both roots Sanskrit — heavily tatsama |
| ಲೋಕಾಡಳಿತ | ಲೋಕ (loka) is Sanskrit; rejected |

### Parallel Formations (the -ಆಳ್ವಿಕೆ pattern)

| English | Kannada | Eke |
|---------|---------|-----|
| democracy | ಮಂದಿಯಾಳ್ವಿಕೆ | mandiyALvike |
| autocracy | ತನ್ನಾಳ್ವಿಕೆ | tannALvike |
| aristocracy | ಮೇಲ್ಮಂದಿಯಾಳ್ವಿಕೆ | mElmandiyALvike |
| oligarchy | ಕೆಲಮಂದಿಯಾಳ್ವಿಕೆ | kelamandiyALvike |

**Summary**: ಮಂದಿಯಾಳ್ವಿಕೆ (mandiyALvike) is built entirely from Dravidian roots — ಮಂದಿ (people, folk) and ಆಳ್ವಿಕೆ (governance, from verb ಆಳು) — joined by regular Kannada vowel-sandhi. No Sanskrit at any level.
