# Native Kannada Word for 'Democracy' — DNS Bhat Style (Baseline)

**Proposed word**: ಎಲ್ಲರಾಳ್ವಿಕೆ (*ellaraaLvike*)

- **ಎಲ್ಲರ** (*ellara*) = genitive of *ellaru* (everyone) — native Kannada; cognate with Tamil *ellaar*
- **ಆಳ್ವಿಕೆ** (*aLvike*) = noun from verb ಆಳು (to rule) — native Kannada

Literally: "the governance of all / by all"

**Why not others:**
- ಜನಾಡಳಿತ: *jana* is Sanskrit — rejected
- ಲೋಕಶಾಹಿ: *loka* is Sanskrit, *shaahi* is Persian — doubly rejected
- ಪ್ರಜಾತಂತ್ರ: pure Sanskrit compound

**Also viable**: ಎಲ್ಲರ ಆಡಳಿತ (*ellara aaDaLita*) — a two-word form using the same native roots.

The best DNS Bhat-style coinage is **ಎಲ್ಲರಾಳ್ವಿಕೆ** (*ellaraaLvike*): built entirely from the oldest native Kannada stock, immediately parseable by any Kannada speaker.
