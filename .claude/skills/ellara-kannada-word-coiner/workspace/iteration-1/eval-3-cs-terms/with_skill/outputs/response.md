# Dravidian Kannada Words for Computer Science Terms (With Skill)

## Quick Reference Table

| English | Kannada | Eke | Breakdown |
|---------|---------|-----|-----------|
| software | ನಡಸುಬರಹ | naDasubaraha | ನಡಸು (drive/run) + ಬರಹ (writing/text) |
| programmer | ಎಣಿಕೆಬರೆಗ | eNikebarega | ಎಣಿಕೆ (computing) + ಬರೆ (write) + -ಗ (agent) |
| algorithm | ಹೆಜ್ಜೆಸಾಲು | hejjesAlu | ಹೆಜ್ಜೆ (step) + ಸಾಲು (row/sequence) |
| database | ತಿಳಿವುಕಣಜ | tiLivukaNaja | ತಿಳಿವು (knowledge/data) + ಕಣಜ (granary/storehouse) |
| artificial intelligence | ಕಟ್ಟರಿವು | kaTTarivu | ಕಟ್ಟು (build/construct) + ಅರಿವು (awareness/intelligence) |

**Sample tech article sentences:**
- ಈ ನಡಸುಬರಹ ತೆರೆದ ಬಾಗಿಲಿನದು — This software is open-source.
- ಆ ಎಣಿಕೆಬರೆಗ ಹೊಸ ಹೆಜ್ಜೆಸಾಲನ್ನು ಬರೆದ — That programmer wrote a new algorithm.
- ತಿಳಿವುಕಣಜದಲ್ಲಿ ಹತ್ತು ಲಕ್ಷ ಸಾಲುಗಳಿವೆ — The database has ten lakh records.
- ಕಟ್ಟರಿವು ಇಂದು ಮಂದಿಯ ಬದುಕನ್ನು ಬದಲಿಸುತ್ತಿದೆ — Artificial intelligence is changing people's lives today.
