# Dravidian Kannada Words for Computer Science Terms (Baseline)

## Summary Table

| English Term | Kannada Coinage | Pronunciation | Etymology |
|---|---|---|---|
| Software | ತಂತ್ರಾಂಶ | Tantramsha | ತಂತ್ರ + ಅಂಶ |
| Programmer | ತಂತ್ರಿ / ಸಂಕೇತಕ | Tantri / Samketaka | ತಂತ್ರ + ಇ / ಸಂಕೇತ + ಕ |
| Algorithm | ಕ್ರಮಾವಳಿ | Kramavali | ಕ್ರಮ + ಆವಳಿ |
| Database | ದತ್ತಾಂಕೋಶ | Dattankoshe | ದತ್ತ + ಅಂಕ + ಕೋಶ |
| Artificial Intelligence | ಕೃತಕ ಬುದ್ಧಿಮತ್ತೆ | Kritaka Buddhimatte | ಕೃತಕ + ಬುದ್ಧಿ + ಮತ್ತೆ |

**Notes**: These coinages lean heavily on Sanskrit-naturalized roots (ತಂತ್ರ, ಕ್ರಮ, ಕೃತಕ, ಬುದ್ಧಿ, ಕೋಶ) rather than purely Dravidian ones.
