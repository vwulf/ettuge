# Ellara Kannada Words for Science Terms (With Skill)

## Quick Reference Table

| English | Kannada | Eke | Breakdown |
|---------|---------|-----|-----------|
| biology | ಉಸಿರಿಯರಿಮೆ | usiriyarime | ಉಸಿರಿ (bio/life) + ಅರಿಮೆ (-logy/study-of) |
| telescope | ಗೆಂಟುತೋರ್ಪುಕ | genTutOrpuka | ಗೆಂಟು (tele/far) + ತೋರ್ (-scope/showing) + -ಪುಕ (instrument) |
| thermometer | ಬಿಸಿಯಳಕ | bisiyaLaka | ಬಿಸಿ (thermo/heat) + ಅಳಕ (-meter/measurer) |
| astronaut | ಬಾನಹಾಯ್ಗ | bAnahAyga | ಬಾನ್ (astro/sky) + ಹಾಯ್ಗ (-naut/navigator) |
| photograph | ಬೆಳಕುಬರಹ | beLakubaraha | ಬೆಳಕು (photo/light) + ಬರಹ (-graph/writing) |

All five words use the same neo-classical root mapping approach — each Greek/Latin element has a direct native Dravidian equivalent. The method bypasses Sanskrit entirely.
